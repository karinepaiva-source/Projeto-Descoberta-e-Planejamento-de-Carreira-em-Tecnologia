# 🚀 Projeto: Descoberta e Planejamento de Carreira em Tecnologia

## 📌 Sobre o Projeto

Este projeto documenta um fluxo completo de transição de carreira para a área de tecnologia, com foco em **Análise de Dados aplicada à área jurídica**.

A proposta é demonstrar, de forma estruturada, o planejamento estratégico, desenvolvimento de habilidades e construção de portfólio voltados para o primeiro emprego na área tech.

A solução foi construída utilizando prompts do repositório oficial da DIO:
https://github.com/digitalinnovationone/copilot-prompts

---

## 🎯 Objetivo

Realizar a transição da área jurídica para tecnologia, desenvolvendo competências práticas em:

* Excel Avançado
* Power BI
* Python para análise de dados
* Inteligência Artificial aplicada

---

## 🧭 Roadmap de Aprendizado (90 dias)

### 🔹 Fase 1 (Dias 1–30): Fundamentos

* Excel Avançado
* Lógica de dados
* Introdução ao Power BI

### 🔹 Fase 2 (Dias 31–60): Análise de Dados

* Power BI (Dashboards)
* Python básico (Pandas)
* Limpeza e organização de dados

### 🔹 Fase 3 (Dias 61–90): Aplicação prática

* Projeto com dados jurídicos
* Criação de dashboard
* Introdução à IA aplicada
